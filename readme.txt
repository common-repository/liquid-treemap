== Description ==

Visualize the performance of the content in with TreeMap.
Supports to number of comments and pageviews (Required 'WordPress Popular Posts').
The 'TreeMap' menu is added to WordPress's 'Tools' menu.

You can also check the latest information on [LIQUID PRESS](https://lqd.jp/wp/).

= Recommended Plugins =
* [WordPress Popular Posts](https://wordpress.org/plugins/wordpress-popular-posts/) - You can display a TreeMap of the pageviews.

== Installation ==

1. Upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
